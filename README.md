# Docmee

Clean-slate rebuild. An AI chatbot platform for medical clinics in Guatemala,
delivered via WhatsApp Business API — multi-tenant, self-hosted.

This repository starts from zero and is built phase-by-phase per the canonical
build specification. No product code has been written yet.

> The earlier prototype lives in the `Docmee-Alpha` repository (kept as an
> archive). This is a fresh, independent build.
